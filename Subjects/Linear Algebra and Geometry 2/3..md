# The Matrix Case
## 
### 
If ${} \lambda \in \sigma(A) {}$ then ${} V_{\lambda}=\ker (A-\lambda I)\neq \{ 0 \} {}$, so $A-\lambda I {}$ is not invertible. Therefore, ${} \det(A-\lambda I)=0 {}$
